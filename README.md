## 📌 Overview

The **Kitchen Inventory Monitoring System** is an IoT-based solution that automates kitchen inventory tracking using an **ESP32 microcontroller**, **ultrasonic sensors**, and a **16x2 LCD display**. The system monitors item levels in real time and sends data to the **Blynk mobile application** for remote access and notifications.

To support smart purchasing decisions, the system also uses **web scraping** to retrieve current prices and offers for low-stock items.

## 🚀 Features

- Real-time inventory monitoring
- Low-stock alerts
- LCD display integration
- Blynk mobile app connectivity
- Automatic basket opening using a touch sensor and servo motor
- Price and offer retrieval through web scraping

## 🛠️ Technologies Used

- ESP32
- Arduino IDE
- Python
- Blynk IoT Platform
- Ultrasonic Sensors (HC-SR04)
- LCD I2C Display
- Web Scraping
